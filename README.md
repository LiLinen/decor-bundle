# Decor bundle

Symfony bundle for LiLinen/Decor library

## Installation

`composer require lilinen/decor-bundle`

Register the bundle:
```php
<?php
// config/bundles.php

return [
    // ...
    LiLinen\DecorBundle\LiLinenDecorBundle::class => ['all' => true],
];
```
